Battle moderator commands

* Numbers indicate the reason for blocking. The table shows how to {reason}:

  • 0 - standard reason;
  • 1 - nickname;
  • 2 - for using cheats;
  • 3 - account transfer;
  • 4 - powerleveling;
  • 5 - bug usage;
  • 6 - sabotage;

* Make sure that both of these commands, /rbattle and /fbattle, are written without a #/battle= in the link, otherwise they won't work!

Command Description Usage
/w {nickname} {reason} Chat warn for false complaint (The reason is specified manually) /w @username False Complaint
/kick {nickname} We use that to kick player from the battle /kick @username
/blockhour {nickname} {reason} Block player for a hour /blockhour @username 0
/blockday {nickname} {reason} Block player for a day /blockday @username 0
/blockweek {nickname} {reason} Block player for a week /blockweek @username 0
/blockmonth {nickname} {reason} Block player for a month /blockmonth @username 0
/blockgameforever {nickname} {reason} Block player FOREVER /blockgameforever @username 2
/unblockgame {nickname} Unblock player's account /unblockgame @username
/unbanip {ip/nickname} Unban player's IP /unbanip 192.168.1.1/username
/karma {nickname} Check player's karma /karma @username
/battleinfo {battle-url} Information about battle creator /battleinfo 2aaaaab461adc636
/search {nickname} Check if player is online and provides battle like (NOTE: command works only in lobby chat) /search @username
/fbattle {battle-url} Command finishes battle (Example: /fbattle 2aaaaab461adc636) /fbattle 2aaaaab461adc636
/rbattle {battle-url} Command deletes battle (Example: /rbattle 2aaaaab461adc636) /rbattle 2aaaaab461adc636
/mults {nickname} Checks alternative (mult) accounts of player /mults @username
/banip {ip/nickname} IP bans account /banip 192.168.1.1/username
/banminutes {nickname} [False Complaint] We use that in case that player keeps using /vote command without reason, it chat bans player for 5 minutes /banminutes @username False Complaint
/banhour {nickname} {reason} We use that in case that player keeps using /vote command without reason, it chat bans player for one hour /banhour @username False Complaint
/comps Checks complaints (votes) in battle /comps
/done Close complaint (The moderator's review of complaints has been completed) /done

FAQ

How to identify a cheater?

Any clear evidence of player cheating leads to eternal lockout.

The category of clear evidence includes:

  • Fly hack/jump hack

    • Most often this can be found on parkour battles or holiday battles, where players most often use this cheat to catch gold boxes. One clean moment is enough to block a player FOREVER!

  • Auto aim

    • Mostly used during parkour fights for striker turret. Key moments for detecting this hack: Following player's striker laser - if player managed to shot 4 rockets without aiming at enemy - it is considered as hack. For this kind of hack evidence, it is better to have multiple clips of it. This hack leads to the FOREVER BLOCK!

  • Unflip hack

    • As in the case of Fly hack, one clear moment is enough to block the player FOREVER! The moment can be understood as the moment when the player's tank is completely flipped, and he manages to turn over himself without getting any impact from his turret/opponent's turret.

  • Speed cheat

    • Speed hack - a type of hack which drastically increases your movement speed. There are many kinds of speed hacks, some of them are easily detected (player's tank is as fast as with crisis drone), but some of them are kindly harder. The best way how to spot a speed hacker is to follow his tank. An example: if the player uses hornet m3, and he is as fast (or even faster) as the user who plays with wasp m3, then it is enough evidence to block him. For that hack we block user FOREVER!

  • No recoil

    • It is used mainly in XP/BP mode. This cheat can be easily detected by watching the player's railgun shots. Any impact the hacker receives from the player has no effect on his aim or tank. For this type of hack, it is best to take a multiple clips of player. For no recoil hack, we block FOREVER.

  • Vertical hack

    • Used mostly in XP/BP battles. How to spot this hack: You should follow player's shots, if he hits enemy from higher places without leaning down his tank then it is considered as hack. A good example are Sandbox/Garder maps, in which it is impossible to hit a player who stands under the wall without leaning down your tank. For this kind of hack, it is better to collect multiple moments. This hack leads to FOREVER block of account.

  • Ignore textures

    • Action in which the player drives through tanks belongs to the category ignore textures. In some cases, it might be hard to detect if a player uses cheats or his ping is too high, that's why we need to gather multiple moments before blocking a user. For this kind of hack, we block user for a MONTH!

  • No reload

    • This hack is mostly used in parkour battles. It's easily noticeable, because player shots constantly without reloading. For this hack, we block player FOREVER!

  • Gravity hack

    • Gravity hack is similar to jump hack, but with small differences. It involves reducing or eliminating the effects of gravity, allowing the player to jump higher, fall slower, or even fly. For that kind of hack, we block player FOREVER.
